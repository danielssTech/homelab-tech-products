# BACKUP & RESTORE - TechProducts Project
Date: 2025-10-05  
Backup location: /opt/backups/

---

## 1. Manual database backup
Command executed:
sudo -u postgres pg_dump -d tech_products -F c -f /opt/backups/techproducts_2025-10-05_14-23.dump

Result:
✔ Backup file successfully created in /opt/backups/  
✔ Verified contents with `pg_restore -l`  
✔ File owned by postgres:postgres  

---

## 2. Automated daily backup (cron job)
User: postgres  
Scheduled task:
0 2 * * * pg_dump -d tech_products -F c -f /opt/backups/techproducts_$(date +\%F_\%H-%M).dump

Frequency: Every day at 02:00 AM  
Verification: `sudo crontab -u postgres -l`  

---

## 3. Database restoration test
Database created:
sudo -u postgres createdb techproducts_restore  

Restore command:
sudo -u postgres pg_restore -O -x -d techproducts_restore /opt/backups/techproducts_2025-10-05_14-23.dump
